Description

PDelay is a lightweight delay/sleep utility designed for batch scripts and automation workflows. It allows you to pause script execution for a specified duration with millisecond precision.

Platform: Windows 10/11 (x64)
License: Freeware
Version: 1.0.0

Features

  • Millisecond precision timing
  • Support for seconds, minutes, and hours
  • Minimal resource usage
  • No installation required
  • Silent operation - no console output

Usage

PDelay.exe [duration] [unit]

duration - The delay duration (numeric value)
unit - Time unit: ms (milliseconds), s (seconds), m (minutes)

Examples

PDelay.exe 500 ms - Wait 500 milliseconds
PDelay.exe 5 s - Wait 5 seconds
PDelay.exe 2 m - Wait 2 minutes
PDelay.exe 1500 - Wait 1500 milliseconds (default unit)
